1) Задача о Рюкзаке


**Задача о рюкзаке** (англ. Knapsack problem) — дано **N** предметов, **n[i]** предмет имеет массу **w[i] > 0** и стоимость **$$p_i$$ > 0**. Необходимо выбрать из этих предметов такой набор, чтобы суммарная масса не превосходила заданной величины **W** (вместимость рюкзака), а суммарная стоимость была максимальна.

Есть несколько модификаций этой задачи, рассмотрим некоторые из них в порядке возрастания сложности
